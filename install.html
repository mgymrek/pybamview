Pybamview

Browser-based application for viewing bam alignments

{% include menu.html %}

Note: PyBamView requires Python2.6.

easy installation

The easiest and preferred way to install PyBamView is using pip: {% highlight bash %} pip install pybamview {% endhighlight %} This will install the latest stable PyBamView version listed in PyPi, and any missing libraries it depends on.

To use the "Export to PDF" feature, you will also need to install rsvg-convert: {% highlight bash %} sudo apt-get install librsvg2-bin {% endhighlight %} (note that the "sudo apt-get" command is assuming you're using Ubuntu/Debian. If you're using a different system you will need to use that system's package manager.)

installing from source

requirements

First, make sure you have the required python packages installed:
  • Flask
  • pyfasta
  • pysam
You can install these using easy_install: {% highlight bash %} sudo easy_install Flask pandas pyfasta pysam {% endhighlight %} If you want to use the "Download PDF" feature, you must have rsvg-convert installed (see above).

getting the source code

Obtain the source code by either:
  • Downloading the tarball from github. {% highlight bash %} tar -xzvf mgymrek-pybamview-[version].tar.gz cd mgymrek-pybamview-[version] {% endhighlight %}
  • Cloning the git repository {% highlight bash %} git clone https://github.com/mgymrek/pybamview.git cd pybamview {% endhighlight %}

building and installing

At the top of the PyBamView source directory, type: {% highlight bash %} sudo python setup.py install {% endhighlight %} To make sure it was installed, type: {% highlight bash %} pybamview --help {% endhighlight %} and ensure that the help message comes up. Then move on to the usage page to get started!
